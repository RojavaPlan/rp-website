Learn the Kurdish Language

The main languages spoken in Rojava are Arabic (most widely spoken), Kurdish (widely used) and Turkish. Without being able to speak one of these languages, you will be unable to work here effectively and only be given limited roles under supervision.

Learning Kurdish is far easier than Arabic, and you’ll be able to communicate with the majority of people you interact with. We suggest simply learning Kurdish which is widely understood.

To maximise your potential, we suggest that people who arrive here take the time to learn Kurdish. When here, we'll help you learn Kurdish with courses and placements. But before you arrive here, you can begin your preparations. The effort you put in will help you before you arrive, but don’t be put off by not seeing progress. The grammar you memorise will help towards picking up the language once you learn the words in Rojava.

First we recommend you bring a laptop which will enable you to work, read documents and so on. A small phone for loading the WQFerheng dictionary (see dictionaries section below) is very useful too. A tablet is less effective, but if you have one you can also bring it. An ebook reader is also a good idea if you plan to go to the war front as their battery lasts a long time.

You can right click these documents, and select “Save Link As…” to download them on your computer.

  • The recommended study book is Kurmanji basic and required reading before coming here. It will only teach you the basic grammar, as you will easily pick up vocabulary here.
  • For intermediate level, study “Kurmanji Kurdish For The Beginners”.
  • The book Learn KURDISH by Baran Rizgar is comprehensive with lots of examples and useful info you won’t find in other books.
  • Another useful book is “Em hînî Kurmancî dibin”.
  • Here is a Kurdish grammar book for EBook readers.
  • This is a good reference book on Kurdish grammar.
  • The website LearnKurmanji is a great starting resource with lots of video and lessons for beginners.

Anki flashcards. To use them, follow these steps:

  • Install Anki (Linux):
    1. Download. Save this file.
    2. Install. Go to where you saved the file and type: sudo dpkg -i anki-2.0.33.deb
    3. Run. Type: anki
  • Install Anki (other OS):
    1. Follow the instructions on Anki's website.
  • Use the cards:
    1. Download and save the cards.
    2. Run Anki, click on 'Import file' and import the cards.
    3. Click on one of the cards, and start studying!
